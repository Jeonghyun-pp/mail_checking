# mailchecking — 최종 기능 명세

Snov.io와 동등한 아웃리치 플랫폼을 목표로 하는 전체 기능 목록.
✅ = 완료, 🔨 = 진행 예정, 빈칸 = 계획.

## Phase 1 — 리드 발굴 & 검증

### 이메일 발굴 (Email Finder)
- ✅ 이름 + 도메인 기반 이메일 패턴 추측·검증
- ✅ 후보 패턴 12종 생성, 신뢰도 스코어링
- ✅ 대량 발굴 (CSV 입력, 백그라운드 처리)
- 도메인 검색 — 회사 도메인으로 소속 이메일 전체 수집
- 회사명/직책/업종 기반 검색
- B2B 리드 데이터베이스 (수집된 리드 누적 검색)
- LinkedIn 프로필에서 이메일 발굴

### 이메일 검증 (Email Verifier)
- ✅ 7단계 검증: 문법 → MX → 일회용 → 역할기반 → SMTP 프로브 → 캐치올 → 스코어
- ✅ 단건 + 대량 검증 (최대 1만 건)
- ✅ 결과 CSV 내보내기
- 검증 결과 캐싱 (중복 검증 비용 절감)
- Greylisting 재시도 처리

## Phase 2 — 콜드메일 캠페인

- ✅ 발신 메일박스 연결 (SMTP, 연결 테스트) — SPF·DKIM·DMARC 점검은 미구현
- ✅ 드립 캠페인 빌더 — 다단계 시퀀스, 단계별 지연
- ✅ BullMQ 발송 워커 (60초 틱, 일일 한도)
- ✅ 오픈 추적 (트래킹 픽셀)
- ✅ 클릭 추적 (링크 래핑 + 리다이렉트)
- ✅ 이메일 개인화 변수 ({{firstName}} {{company}} 등)
- ✅ 캠페인 분석 (발송·오픈·클릭·답장·바운스 집계)
- ✅ 바운스 시 수신자 자동 제외
- ✅ 답장 감지 (IMAP 폴링, 5분 주기) → 수신자 REPLIED·시퀀스 중단
- ✅ 수신거부(unsubscribe) 링크 + 자동 푸터·전역 차단
- ✅ 텍스트 스피닝 ({옵션1|옵션2|옵션3} 랜덤 치환)
- ✅ A/B 테스트 (스텝별 제목·본문 변형, 50/50 분할, 변형별 분석)
- 조건 분기 트리거 (오픈/클릭 시 분기) — 캠페인 모델 그래프화 필요, 미착수
- 멀티채널 — 이메일 + LinkedIn 단계 혼합
- 발송 시간대 최적화 / 수신자 타임존

## Phase 3 — CRM & 파이프라인

- ✅ 파이프라인 다중 생성 (기본 5단계 자동 생성)
- ✅ 딜 카드 — 드래그앤드롭 단계 이동 (칸반 보드)
- ✅ 리드 ↔ 딜 연결
- ✅ 태스크/할 일 관리, 마감일, 완료 토글
- ✅ 활동 타임라인 (노트·통화·이메일·미팅 + 단계이동 자동 기록)
- 파이프라인 단계 커스터마이즈 (이름·순서 편집 UI) — 미구현
- Google Calendar 동기화 — OAuth 필요, 미착수
- 첨부파일 — 파일 저장소 필요, 미착수

## Phase 4 — 도달률 & 발신자 평판

- ✅ 이메일 웜업 — 램프업 스케줄(+2/일), 발송 틱, 일일 쿼터
- ✅ 웜업 풀 (계정 간 자동 주고받기) + 웜업 통계 페이지
- ✅ 웜업 IMAP 수신 처리 — 자동 열람·답장·스팸→인박스 이동 (실 IMAP 메일함 필요)
- ✅ 도달률 테스트 — SPF·DKIM·DMARC·MX 점검 + 스팸 콘텐츠 분석 → 점수
- 도달률 테스트(인박스/스팸/프로모션 시드 메일함 진단) — 시드 인프라 필요, 미구현
- 도메인 블랙리스트(DNSBL) 모니터링 — 미구현

## Phase 5 — 확장 & 통합

- ✅ Chrome 확장 프로그램 (팝업 + LinkedIn 인페이지 패널, 발굴·검증·저장)
- ✅ 공개 REST API — /api/v1 verify · find · leads
- ✅ MCP 서버 — 28개 도구, 팀원이 자기 Claude Code/Codex로 앱 구동
- ✅ API 키 관리 + 사용량 제한 (일일 레이트 리밋)
- ✅ 웹훅 (LEAD_CREATED · CAMPAIGN_REPLIED · DEAL_WON)
- ✅ CSV 가져오기/내보내기 (리드)
- 서드파티 통합 — HubSpot/Salesforce 등 OAuth 통합은 미구현 (웹훅으로 대체 가능)

## 플랫폼 공통 (전 단계 지원 기능)

### 계정 & 인증 (내부 팀 도구)
- ✅ 회원가입/로그인 — scrypt 비밀번호 해싱, HMAC 서명 세션 쿠키
- ✅ 미들웨어(proxy) 라우트 보호
- ✅ 공유 워크스페이스 — 팀 전원이 같은 리드·캠페인·CRM·메일박스 공유
- ✅ 초대 코드 가입 — 관리자(ADMIN)가 발급한 1회용 코드로만 가입
- ✅ 역할 — ADMIN / MEMBER (초대 코드 발급은 ADMIN 전용)
- 비밀번호 재설정, 2FA — 미구현 (필요 시 추가)
